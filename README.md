# 🎭 fb-emoji

菲比 Bot 表情包分类库，配合 [astrbot_plugin_stealer](https://github.com/nagatoquin33/astrbot_plugin_stealer) 使用。

## 分类目录

| 分类 | 说明 |
|------|------|
| angry | 生气 |
| confused | 困惑 |
| cry | 哭泣 |
| disgust | 嫌弃 |
| dumb | 无语/呆 |
| embarrassed | 尴尬 |
| excitement | 兴奋 |
| fear | 害怕 |
| happy | 开心 |
| love | 喜爱 |
| sad | 难过 |
| shy | 害羞 |
| sigh | 叹气 |
| surprised | 惊讶 |
| thank | 感谢 |
| tired | 疲惫 |
| troll | 搞怪/阴阳 |

## 使用方式

将本仓库 clone 到 AstrBot 的 `plugin_data/astrbot_plugin_stealer/categories/` 目录下，或通过 WebUI 批量导入。
